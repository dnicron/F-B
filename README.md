# Museum of Natural Artifacts

Адаптивный лендинг музея природных артефактов, выполненный по макету из Figma.

## О проекте

Проект представляет собой лендинг с адаптивной версткой и интерактивным слайдером карточек.
Основной упор сделан на семантическую HTML-разметку, адаптивность интерфейса и чистую организацию CSS и JavaScript-кода.

## Функциональность

- Адаптивная верстка для desktop, tablet и mobile устройств
- Семантическая HTML-разметка
- Слайдер карточек на чистом JavaScript
- Интерактивные состояния элементов (hover, focus)
- SVG-иконки и графика
- Flexbox и CSS Grid для построения макета
- БЭМ-нейминг классов

## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Flexbox
- CSS Grid
- SVG

## Что было реализовано

- Header с адаптивной навигацией
- Hero-секция
- Секция карточек со слайдером
- Форма обратной связи
- Footer с социальными ссылками

## Запуск проекта

- Клонировать репозиторий:
  git clone <repository-url>
- Открыть проект в VS Code.
- Запустить через Live Server.
